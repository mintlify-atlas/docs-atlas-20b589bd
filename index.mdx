The **Capinetta RP Bot System** is a production-ready, self-hosted Discord management platform built specifically for FiveM and GTA Roleplay communities. It ships two independent bots — a General Bot for moderation, tickets, and auditing, and a Whitelist Bot for application management — orchestrated by PM2 and backed by MariaDB via Prisma ORM.

## Getting Started

<Steps>
  <Step title="Create two Discord bots">
    Visit the [Discord Developer Portal](https://discord.com/developers/applications), create two applications (General and Whitelist), and copy their tokens.
  </Step>
  <Step title="Configure your environment">
    Copy `.env.example` to `.env` and fill in your bot tokens, MariaDB credentials, and OAuth2 details. See the [Configuration guide](/configuration) for every variable.
  </Step>
  <Step title="Initialize the database and deploy commands">
    Run `npm install`, `npx prisma db push`, then `npm run deploy` to push slash commands to Discord.
  </Step>
  <Step title="Start with PM2">
    Run `npm run prod` to start both bots under PM2 process management. Then run `/setup` in your Discord server to complete configuration.
  </Step>
</Steps>

<Note>
  The system requires **Node.js v18+**, **MariaDB/MySQL 8.0+**, and two separate Discord bot tokens. Redis is optional but recommended for dashboard caching.
</Note>
